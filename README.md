## tcUdec

# Installation

```bash
git clone https://github.com/Lagushka1984/tcUdec.git
cd tcUdec
bash install.sh
```

# Launch
```bash
python3 main.py config.txt
```

# Config file

### Default command template
```bash
cmd=param=name
```

### debug
Description:
Show frames, output in terminal

Template:
```bash
debug=True/False=None
```

Example:
```bash
debug=True=None
```
